### arrayMove(array, oldIndex, newIndex)

_@creuna/utils/array-move_

* `array`: array
* `oldIndex`: number
* `newIndex`: number
* returns: array

Returns a new array that is a deep clone of the one passed in with the item at `oldIndex` moved to `newIndex`.

```js
arrayInsert(["a", "b", "c"], 1, 2); // ['a', 'c', 'b']
```
